# 字幕宽度自适应优化计划

## 问题分析
当前 `SubtitleDisplay.vue` 组件中，字幕宽度受限于：
- `.subtitle-display` 设置 `max-width: 95%` - 限制字幕最大宽度
- `.subtitle-text` 设置 `min-width: 100px` - 固定最小宽度

这导致字幕不能根据实际内容长度自适应宽度。

## 修改方案

### 修改文件：`d:\files\mybilibili-next\mybilibili-web\src\components\SubtitleDisplay.vue`

#### 1. 调整 `.subtitle-display` 容器样式
**位置**：第152-161行

**修改内容**：
- 将 `max-width: 95%` 改为 `max-width: 90%`（稍微收紧，给屏幕边缘留更多空间）
- 保持居中对齐 `text-align: center`
- 移除不必要的宽度限制，让容器紧贴内容

**修改前**：
```css
.subtitle-display {
  position: relative;
  display: inline-block;
  z-index: 100;
  pointer-events: auto;
  max-width: 95%;
  text-align: center;
  user-select: none;
}
```

**修改后**：
```css
.subtitle-display {
  position: relative;
  display: inline-block;
  z-index: 100;
  pointer-events: auto;
  max-width: 90%;
  text-align: center;
  user-select: none;
}
```

#### 2. 调整 `.subtitle-text` 文本样式
**位置**：第167-173行

**修改内容**：
- 移除 `min-width: 100px` 的固定最小宽度限制
- 保留 `word-wrap: break-word` 以处理长单词换行
- 添加 `max-width: 100%` 确保文本不会溢出容器

**修改前**：
```css
.subtitle-text {
  display: inline-block;
  cursor: move;
  transition: opacity 0.2s ease;
  word-wrap: break-word;
  min-width: 100px;
}
```

**修改后**：
```css
.subtitle-text {
  display: inline-block;
  cursor: move;
  transition: opacity 0.2s ease;
  word-wrap: break-word;
  max-width: 100%;
}
```

## 预期效果
- 字幕容器会根据字幕内容实际宽度自适应
- 短字幕会显示紧凑，不会被撑到固定最小宽度
- 长字幕会自动换行，不会超出屏幕
- 保留最大宽度限制，防止字幕过宽影响观看
