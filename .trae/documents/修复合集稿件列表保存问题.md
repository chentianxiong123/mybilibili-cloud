# 修复合集创建时稿件列表无法保存的问题

## 问题分析

根据日志分析：
- 合集可以成功创建
- 但稿件列表为空 (`Array(0)`)
- 数据库中 `manuscript_collection_relations` 表没有对应记录

**根本原因**：`ManuscriptCollectionRelationMapper.insert()` 方法缺少 `@Insert` 注解，导致插入操作未真正执行。

## 修复步骤

### 1. 修复 ManuscriptCollectionRelationMapper.java
- 为 `insert` 方法添加 `@Insert` 注解

### 2. 修复 ManuscriptCollectionMapper.java
- 为 `insert` 方法添加 `@Insert` 注解
- 为 `update` 方法添加 `@Update` 注解

### 3. 重新编译
- `mvn compile -pl mybilibili-interaction -am -DskipTests`

### 4. 重启服务测试
- 重启 `mybilibili-interaction` 服务
- 创建新合集并选择稿件
- 验证数据库中是否有对应关联记录
